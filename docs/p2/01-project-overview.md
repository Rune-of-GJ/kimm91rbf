# SpeakFlow P2 — 프로젝트 개요

## 프로젝트 비전

**SpeakFlow P2**는 P1의 학생 중심 강의 플랫폼을 넘어,  
**역할 기반 운영 + 월정액 학습 + 1:1 첨삭**이 결합된 스피치 훈련 플랫폼으로 확장하는 단계다.

P2에서 SpeakFlow는 단순 LMS가 아니라 다음 구조를 갖는다.

- 학생: 월정액으로 강의를 시청하고 첨삭을 요청한다
- 강사: 강의를 직접 등록하고 첨삭 피드백을 제공한다
- 관리자: 사용자와 역할을 관리한다

---

## 핵심 가치

| 가치 | 설명 |
|------|------|
| 역할 분리 | `student / instructor / admin` 권한 체계 |
| 훈련 중심 | 강의 소비보다 반복 연습과 피드백 흐름 강조 |
| 구독 모델 | 강의 시청은 월정액 기반으로 전환 |
| SpeakFlow 전용 기능 | 1:1 첨삭 요청과 강사 피드백 루프 |

---

## 서비스 대상

### 학생 (student)

- 강의 탐색
- 멤버십 가입
- 강의 시청
- 첨삭권 구매
- 첨삭 요청 / 결과 확인

### 강사 (instructor)

- 강의 등록
- 강의편 추가/수정
- 첨삭 요청 대기열 확인
- 총평/타임코드 피드백 작성

### 관리자 (admin)

- 최초 관리자 계정 생성
- 사용자 목록 조회
- 역할 변경
- 사용자 삭제

## 현재 P2의 서비스 정의

현재 SpeakFlow P2는 아래 문장으로 정리할 수 있다.

> SpeakFlow는 월정액으로 강의를 학습하고, 필요할 때 1:1 첨삭을 추가 구매하거나 Coach 플랜 포함 혜택으로 받아볼 수 있는 스피치 훈련 플랫폼이다.
