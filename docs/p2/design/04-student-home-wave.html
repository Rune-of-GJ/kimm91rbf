What this version changes from the previous example

The first set was mostly system-driven. This version is still controlled, but it has a more recognizable brand move thanks to the logo mark and the wave expansion pattern.

Visual logic
  • The hero image area becomes darker so the lime pulse reads more clearly
  • Curve outlines act like ripples instead of generic decorative circles
  • The white section still begins as one large rounded surface below the hero
  • The same circular expansion is now visible in stats, programs, reviews, and footer
Brand logic
  • The smile-tube form can evolve into the actual platform logo system
  • Black and lime are now tied to the mark instead of floating independently
  • The motion cue is simple enough to reuse in future sections and dashboards
  • The whole page now feels driven by one recognizable shape rather than isolated decoration
