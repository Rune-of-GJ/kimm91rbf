# P1 기획서

## 프로젝트 개요

본 프로젝트는 발표 능력 및 커뮤니케이션 역량 향상을 위한 온라인 스피치 강의 플랫폼을 개발하는 것을 목표로 한다.

사용자는 다양한 스피치 강의를 탐색하고 수강할 수 있으며, 강의를 통해 발표 기술, 면접 스킬, 설득 화법 등 말하기 능력을 향상할 수 있다.

프로젝트는 단계적으로 확장되는 구조로 설계되며, P1 단계에서는 학생 중심의 MVP 기능을 구현한다.

## 서비스 대상

- 발표 능력을 향상시키고 싶은 사용자
- 면접 준비생
- 커뮤니케이션 능력을 향상시키고 싶은 직장인
- 스피치 강의를 제공하는 강사

## 성공 지표 (KPI)

- 월간 활성 사용자(MAU)
- 강의 수강 신청 전환율(강의 상세 조회 -> 수강 신청)
- 강의 완료율(수강 시작자 대비 완강자 비율)
- 사용자 만족도(간단한 피드백 설문)

## 범위 및 가정

- P1은 인증(이메일/비밀번호), 강의 목록/상세/수강/진도 저장까지를 포함한다.
- P1 인증은 세션 기반으로 구현한다.
- 결제, 관리자/강사 전용 대시보드, 알림 등은 P2/P3에서 구현된다.
- 동영상은 YouTube 링크를 사용한다.

## 위험요소 및 대응

- 동영상 접근성(유튜브 링크 차단 등): 대체 영상 호스팅 옵션 검토
- 개인정보 보안: 비밀번호 해시(Bcrypt) 및 HTTPS 환경 권장
- 데이터 모델 변경: 마이그레이션 계획 수립
- 프론트 과구현 위험: DB와 API 응답에 없는 정보는 화면에 임의로 추가하지 않음

## 주요 기능 (P1)

- 강의 탐색: 카테고리별 강의 조회
- 강의 수강 신청: 사용자가 강의를 신청하여 내 강의에 추가
- 강의 시청: YouTube 링크 기반 영상 시청
- 학습 진도 관리: 시청 여부 및 완료 상태 저장

## 서비스 흐름

1. 회원가입 / 로그인
2. 카테고리 선택
3. 강의 목록 조회
4. 강의 상세 조회
5. 수강 신청
6. 강의 시청
7. 학습 진도 관리

## 현재 구현 상태

- 백엔드 P1 핵심 API 구현 완료
- 카테고리 / 강의 / 수강 / 진도 핵심 흐름 테스트 완료
- 프론트는 백엔드 응답과 실제 DB 기준으로만 구성 예정
- 디자인/프론트 작업은 문서와 API 기준 고정 후 진행

## 단계 계획 요약

- P1: 학생 중심 MVP 기능 구현
- P2: 역할 분리(관리자, 강사, 학생) 및 인증 기능 강화
- P3: 결제, 알림, 로깅, 모니터링 등 운영 기능 추가
